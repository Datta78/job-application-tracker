# Job Application Tracker

A simple but powerful tool to track job applications, status, and statistics.

Perfect for job search and career management.

## ✨ Features
- Add new job applications
- Track status (Applied, Interview, Rejected, Offer)
- View all applications with dates
- Basic statistics
- Data saved in JSON file

## 🛠️ Tech Stack
- Python 3
- JSON for data persistence

## 🚀 How to Run
```bash
python main.py
